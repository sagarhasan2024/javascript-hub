JavaScript Inheritance Explained

Understanding prototypal inheritance and class-based syntax in JavaScript

JavaScript Inheritance Models

JavaScript uses prototypal inheritance, where objects can inherit properties and methods from other objects via their prototype chain.

Prototype Chain

When you try to access a property or method on an object, JavaScript first looks at the object itself. If it doesn't find it, it looks at the object's prototype, then the prototype's prototype, and so on up the chain.

Constructor Function Inheritance

// Parent constructor function Animal(name) { this.name = name; } // Parent methods Animal.prototype.eat = function() { return `${this.name} is eating.`; }; Animal.prototype.sleep = function() { return `${this.name} is sleeping.`; }; // Child constructor function Dog(name, breed) { Animal.call(this, name); // Call parent constructor this.breed = breed; } // Set up inheritance Dog.prototype = Object.create(Animal.prototype); Dog.prototype.constructor = Dog; // Child methods Dog.prototype.bark = function() { return `${this.name} is barking!`; }; // Create instance const myDog = new Dog("Buddy", "Golden Retriever");

ES6 Class Inheritance

// Parent class class Animal { constructor(name) { this.name = name; } eat() { return `${this.name} is eating.`; } sleep() { return `${this.name} is sleeping.`; } } // Child class class Dog extends Animal { constructor(name, breed) { super(name); // Call parent constructor this.breed = breed; } bark() { return `${this.name} is barking!`; } } // Create instance const myDog = new Dog("Buddy", "Golden Retriever");

Object.create() Inheritance

// Parent object const animal = { init: function(name) { this.name = name; return this; }, eat: function() { return `${this.name} is eating.`; }, sleep: function() { return `${this.name} is sleeping.`; } }; // Create child object const dog = Object.create(animal); dog.init = function(name, breed) { animal.init.call(this, name); this.breed = breed; return this; }; dog.bark = function() { return `${this.name} is barking!`; }; // Create instance const myDog = Object.create(dog).init("Buddy", "Golden Retriever");
Key Point: Despite the class syntax introduced in ES6, JavaScript still uses prototypal inheritance under the hood. The class syntax is primarily syntactic sugar.

class Animal { constructor(name) { this.name = name; } speak() { return `${this.name} makes a sound.`; } } class Dog extends Animal { speak() { return `${this.name} barks.`; } } class Cat extends Animal { speak() { return `${this.name} meows.`; } }

Example 2: Employee Management System

This example shows a more practical inheritance use case with different employee types.

class Employee { constructor(name, position, salary) { this.name = name; this.position = position; this.salary = salary; } getDetails() { return `${this.name} - ${this.position}`; } getAnnualSalary() { return this.salary * 12; } } class Manager extends Employee { constructor(name, salary, department) { super(name, "Manager", salary); this.department = department; } getDetails() { return `${super.getDetails()} (${this.department})`; } manageTeam() { return `${this.name} is managing the ${this.department} team.`; } } class Developer extends Employee { constructor(name, salary, programmingLanguage) { super(name, "Developer", salary); this.programmingLanguage = programmingLanguage; } getDetails() { return `${super.getDetails()} - ${this.programmingLanguage}`; } writeCode() { return `${this.name} is writing code in ${this.programmingLanguage}.`; } }

Example 3: Method Overriding

This example demonstrates how child classes can override parent methods while still accessing the parent implementation.

class Vehicle { constructor(type, speed) { this.type = type; this.speed = speed; } move() { return `The ${this.type} is moving at ${this.speed} mph.`; } stop() { return `The ${this.type} has stopped.`; } } class Car extends Vehicle { constructor(speed, brand) { super("car", speed); this.brand = brand; } move() { // Call parent method and extend functionality return `${super.move()} The ${this.brand} is on the road!`; } honk() { return `The ${this.brand} is honking!`; } } class Plane extends Vehicle { constructor(speed, airline) { super("plane", speed); this.airline = airline; } move() { // Completely override parent method return `The ${this.airline} plane is flying at ${this.speed} mph.`; } takeOff() { return `The ${this.airline} plane is taking off!`; } }
